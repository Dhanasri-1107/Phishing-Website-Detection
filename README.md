# Phishing Website Detection

## Overview
This project detects whether a website is legitimate or phishing using Machine Learning.

## Features
- Random Forest Classifier
- User-friendly interface
- Predicts phishing websites
- Displays prediction result

## Technologies Used
- Python
- Pandas
- Scikit-learn
- Streamlit (or FastAPI)
- HTML/CSS

## Dataset
Dataset contains various website features used for phishing detection.

## Installation

```bash
pip install -r requirements.txt
```

## Run

```bash
streamlit run app.py
```

or

```bash
python app.py
```
